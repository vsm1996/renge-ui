================================================================================
                          COMMON MISTAKES
================================================================================

❌ WRONG: Hardcoded spacing
  padding: 15px;
  margin: 7px;
  gap: 22px;

✅ RIGHT: Using tokens
  padding: var(--renge-space-4);
  margin: var(--renge-space-3);
  gap: var(--renge-space-6);

❌ WRONG: Arbitrary font sizes
  font-size: 18px;

✅ RIGHT: Using type scale
  font-size: var(--renge-fontSize-lg);  /* 25.9px = 16 × φ */

❌ WRONG: Random container width
  max-width: 1000px;

✅ RIGHT: Using container tokens
  max-width: var(--renge-container-md);  /* 848px = 200 × φ³ */

================================================================================
                         KEY CONCEPTS
================================================================================

FIBONACCI-BASED (Spacing, Radius, Duration):
  • Best for: Discrete steps (4, 8, 12, 20, 32, ...)
  • Used when: "What's the next size that feels proportional?"
  • Pattern: Each value = sum of two previous (1+1=2, 1+2=3, 2+3=5, ...)

PHI-BASED (Container Widths, Type Sizes):
  • Best for: Continuous growth across breakpoints
  • Used when: "How much larger should this be?"
  • Pattern: Each value = φ × previous (exact 1.618 multiplier)

FRACTAL SCALING (Component Self-Similarity):
  • Used when: Small button has same internal logic as large modal
  • Pattern: 4 × φⁿ (exact powers of φ)
  • Rare but critical for recursive UI

MOTION EASING (Smooth Acceleration):
  • All curves use 1/φ² (0.382) and 1/φ (0.618) control points
  • These "golden splits" create natural-feeling curves
  • A + B = 1.000 (perfect complement)

PHYLLOTAXIS (Radial Patterns):
  • Golden angle (137.5°) prevents visual clustering
  • Found in nature: sunflowers, pinecones, plant spirals
