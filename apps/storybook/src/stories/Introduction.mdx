# Renge Design System

Renge is a design system grounded in natural mathematical proportions — Fibonacci sequences, the golden ratio (φ ≈ 1.618), and OKLCH color science.

## Two layers

**`@renge/tokens`** — the proportional source of truth. Generates CSS custom properties for spacing, typography, motion, radius, and color. Zero runtime dependencies.

**`@renge/react`** — optional React component layer. Components use inline styles referencing CSS custom properties; no CSS-in-JS runtime, no class names.

## Color profiles

Three profiles map the same palette to semantic tokens:

| Profile | Character |
|---------|-----------|
| `clear` | Clean light mode, neutral grays |
| `earth` | Warm light mode, organic tones |
| `twilight` | Dark mode, deep backgrounds |

Switch profiles using the **Profile** toolbar dropdown above any story.
